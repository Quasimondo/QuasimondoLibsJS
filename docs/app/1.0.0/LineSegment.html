Class: LineSegment

LineSegment(arg1opt, arg2opt, arg3opt, arg4opt)

new LineSegment(arg1opt, arg2opt, arg3opt, arg4opt)

Represents a line segment defined by two points.
Parameters:
Name Type Attributes Description
arg1 number | qlib.Vector2 <optional>
 If a number, the x-coordinate of the first point. If a Vector2, the first point.
arg2 number | qlib.Vector2 <optional>
 If arg1 is a number, the y-coordinate of the first point. If arg1 is a Vector2, this is the second point (Vector2).
arg3 number | qlib.Vector2 <optional>
 If arg1 and arg2 are numbers, this is the x-coordinate of the second point or the second point itself (Vector2).
arg4 number <optional>
 If arg1, arg2, arg3 are numbers, this is the y-coordinate of the second point.
Properties:
Name Type Description
p1 qlib.Vector2 The first point of the line segment.
p2 qlib.Vector2 The second point of the line segment.
p1_end boolean Indicates if p1 is an endpoint (always true for LineSegment).
p2_end boolean Indicates if p2 is an endpoint (always true for LineSegment).
type string The type of the geometric shape ("LineSegment").
angle number The angle of the line segment in radians.
length number The length of the line segment.
